# Playbook Base

This project provides the base components for the `playbook-postgresql` library. It includes shared classes and interfaces for filtering, exceptions, and other common functionalities.

This library is not intended for direct use, but rather as a foundational dependency for `playbook-postgresql`.

For detailed usage and more information, please refer to the [`playbook-postgresql`](https://github.com/ilidazilv/playbook-postgresql) repository.
